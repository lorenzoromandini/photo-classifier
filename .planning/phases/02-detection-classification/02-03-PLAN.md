    - validateImage(String filePath): Future<ImageValidationResult>
      - Check file exists
      - Check file size > 0
      - Try to read first bytes (corruption detection)
      - Return validation result with error if invalid
    - getImageDimensions(String filePath): Future<Size?>
      - Return original dimensions before compression
    
    Per RESEARCH.md: HEIC requires API 28+, convert to JPEG before ML.
    Per ERR-02: Validate file before processing to avoid ML Kit crashes.
