<task type="auto">
  <name>Task 3: Create TrashCleanupWorker for auto-deletion</name>
  <files>
    app/src/main/java/com/example/photoorganizer/data/worker/TrashCleanupWorker.kt
  </files>
  <action>
    Create daily worker for trash cleanup:

    1. **TrashCleanupWorker class**:
       - Extend CoroutineWorker
       - @HiltWorker with @AssistedInject
       - Inject TrashManager, TrashRepository
       - override suspend fun doWork(): Result

    2. **Worker implementation**:
       ```
       - Get current timestamp
       - Query trashRepository.getExpiredItems(now)
       - For each expired item:
         - Call trashManager.permanentlyDelete(item)
         - If success: remove from database
         - If failure: log and continue (don't block)
       - Return Result.success() with Data containing deletedCount
       ```

    3. **Scheduling**:
       - PeriodicWorkRequest with 24-hour repeat interval
       - Constraints: requires battery not low
       - Initial delay: schedule at app startup after onboarding
       - Unique work name: "trash_cleanup"
       - ExistingPeriodicWorkPolicy: KEEP (don't duplicate)

    4. **Integration**:
       - Schedule from onboarding completion
       - Or from MainActivity.onCreate if already onboarded

    5. **Edge cases**:
       - If trash folder doesn't exist, nothing to do
       - If delete fails, item stays in DB (will retry next day)
       - Handle case where file already deleted manually
  </action>
